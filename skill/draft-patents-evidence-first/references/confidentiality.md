# Confidentiality boundary / 保密边界

Default to `local_only`: no source content, hashes, paths, extracted terms, or queries leave the local environment. For `sanitized_search`, remove people, organizations, project codes, unique dimensions, unpublished parameter combinations, credentials, and local paths; show the sanitized query to a human before use. For `approved_external`, record the scoped data, recipient/service, purpose, approver label, and timestamp.

Any confirmed credential, token, private key, cookie, personal contact detail, absolute machine path, denylist term, hidden document metadata, tracked change, or undeclared archive member blocks packaging. Never upload, publish, file, or perform an irreversible external change.
