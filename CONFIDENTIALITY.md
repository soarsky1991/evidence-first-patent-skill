# Confidentiality policy

The default case mode is `local_only`. It prohibits external requests and keeps source content, hashes, paths, and extracted terms local. `sanitized_search` requires a human-previewed query with identifiers and unpublished parameter combinations removed. `approved_external` requires a scoped record of excerpts, recipient, purpose, approver label, and timestamp. No command files, publishes, uploads, or submits a patent application.
