# Bilingual terminology / 双语术语表

| 中文 | English | Usage note / 使用说明 |
|---|---|---|
| 证据记录 | evidence record | one source-backed record with a location / 一条具有来源和位置的记录 |
| 证据分类 | evidence class | measured, documented, inferred, designed / 实测、已记载、推断、拟议设计 |
| 实测 | measured | requires method/context and source location / 须有方法、背景和来源位置 |
| 已记载 | documented | explicit source statement, not project measurement / 来源明确陈述，但并非项目实测 |
| 推断 | inferred | must list `derived_from`; never observed fact / 须列明推导来源，不得写成观察事实 |
| 设计/拟议 | designed | proposal or target; not an achieved result / 方案或目标，并非已实现结果 |
| 权利要求限定 | claim limitation | one atomic technical feature or relation / 单一原子技术特征或关系 |
| 权利要求溯源 | claim trace | limitation-to-evidence mapping / 限定与证据之间的映射 |
| 人工选择门 | human selection gate | concept and public-boundary confirmation / 对选题和公开边界的人工确认 |
| 最接近现有技术 | closest prior art | comparison unit after family deduplication / 专利族去重后的对比单元 |
| 家族标识 | family identifier | normalized identifier or explicit unknown / 规范标识或明确记为未知 |
| 公开日 | publication date | verified date required for date-qualified comparison / 临界日评价须使用经核验日期 |
| 公开边界 | public boundary | what may leave local-only handling / 可以离开本地处理环境的内容范围 |
| 待验证 | to be verified | preferred wording for target or open question / 目标或待决问题的推荐措辞 |
| 方案 | proposed embodiment | design option, not a test result / 设计选项，并非试验结果 |
