# Social preview visual brief / 社会化预览图说明

## Purpose / 用途

`public-assets/social-preview.jpg` is a neutral GitHub social preview for the repository. It is a generated illustration, not a photograph of a real patent matter, test setup, source document, or measured apparatus.

`public-assets/social-preview.jpg` 是仓库的中性 GitHub 社会化预览图，为合成插图，并非真实专利案件、试验装置、来源文件或实测设备的照片。

## Material text equivalent / 重要文字等价文本

- Evidence-First Patent Skill
- Draft from evidence, not imagination.
- Claim-to-Evidence Trace: 1 Limitation; 2 Source; 3 Evidence; 4 Review.
- Project Notebook; Objective: Secure cylindrical component to round bar.
- Constraints: Design option; To be verified; Proposed target; Optional feature.
- Source Document; Source Location; Evidence Table (to be completed): Item; Location; Description; Notes.
- Illustration labels: lever; pivot; clamp band; bar.
- Synthetic illustration / 合成示意.

## Final generation brief / 最终生成说明

- Use case: `productivity-visual`.
- Composition: wide 2:1 banner with an illustrative engineering notebook, a generic source record, a mechanical-clamp sketch, and a four-step claim-to-evidence trace.
- Style: restrained editorial technical illustration; warm white, graphite, muted teal, and amber verification accents.
- Required text: `Evidence-First Patent Skill`; `Draft from evidence, not imagination.`; `Synthetic illustration / 合成示意`.
- Prohibitions: no company marks, people, names, seals, confidential data, dates, performance claims, test results, hype language, watermarks, or third-party logos.

The first generated candidate was rejected because it contained misspelled microtext and synthetic measurement-like content. A later candidate still used performance-like constraint wording. The accepted revision removed dates, validation claims, numeric result graphics, and performance-like wording; it replaced them with proposal/verification language, a generic source-location table, short trace labels, and an explicit synthetic-illustration label. The accepted image was resized to 1280 × 640 and JPEG-compressed without adding content.

首张候选图因微文字拼写错误及类似实测数据的合成内容而被否决；后一候选仍含有类似性能结论的约束措辞。最终修订稿删除日期、验证结论、数值结果图和性能化措辞，改用拟议/待验证语言、通用来源位置表、简短溯源标签及明确的“合成示意”标签；随后仅进行 1280 × 640 缩放和 JPEG 压缩，未新增画面内容。
